## State & Storage

Hooks for managing complex state, boolean logic, and browser storage (Local Storage, Session Storage, Cookies, IndexedDB).

- [`useBoolean`](/hooks/use-boolean) — Handles boolean state with useful toggling utilities.
- [`useCounter`](/hooks/use-counter) — Manages a counter with increment, decrement, and reset functionality.
- [`useToggle`](/hooks/use-toggle) — Manages a boolean toggle state.
- [`useStep`](/hooks/use-step) — Manages navigation between steps in a multi-step process.
- [`useMap`](/hooks/use-map) — Manages a key-value Map state.
- [`useDebounceValue`](/hooks/use-debounce-value) — Returns a debounced version of a value.
- [`useCookie`](/hooks/use-cookie) — Manages state synchronized with a browser cookie.
- [`useLocalStorage`](/hooks/use-local-storage) — Manages state synchronized with localStorage.
- [`useReadLocalStorage`](/hooks/use-read-local-storage) — Reads values from localStorage safely.
- [`useSessionStorage`](/hooks/use-session-storage) — Manages state synchronized with sessionStorage.
- [`useIndexedDB`](/hooks/use-indexed-db) — Provides an interface to the IndexedDB API.

## Sensors & DOM

Hooks that interact with the DOM, track elements, and monitor user events.

- [`useClickAway`](/hooks/use-click-away) — Triggers a callback when clicking outside an element.
- [`useClickAnyWhere`](/hooks/use-click-any-where) — Handles click events anywhere on the document.
- [`useHover`](/hooks/use-hover) — Tracks whether a DOM element is being hovered over.
- [`useIntersectionObserver`](/hooks/use-intersection-observer) — Tracks element intersection using the Intersection Observer API.
- [`useResizeObserver`](/hooks/use-resize-observer) — Observes element size changes.
- [`useWindowSize`](/hooks/use-window-size) — Tracks the browser window dimensions.
- [`useScreen`](/hooks/use-screen) — Tracks screen properties.
- [`useScrollLock`](/hooks/use-scroll-lock) — Locks and unlocks the body scroll.
- [`useDocumentTitle`](/hooks/use-document-title) — Updates the document title dynamically.
- [`useMediaQuery`](/hooks/use-media-query) — Tracks the state of a CSS media query.

## Media & Device

Hooks for accessing hardware features like audio, video, clipboard, and screen sharing.

- [`useAudioRecorder`](/hooks/use-audio-recorder) — records audio with real-time analysis.
- [`useUserMedia`](/hooks/use-user-media) — Captures audio and video from the user's device.
- [`useScreenShare`](/hooks/use-screen-share) — Captures the user's screen or application window.
- [`useMediaSession`](/hooks/use-media-session) — Interacts with the Media Session API (media controls).
- [`useCopyToClipboard`](/hooks/use-copy-to-clipboard) — Copies text to the clipboard.

## Lifecycle & Timers

Hooks that help manage component lifecycle, timeouts, intervals, and side effects.

- [`useIsMounted`](/hooks/use-is-mounted) — Determines if the component is currently mounted.
- [`useUnmount`](/hooks/use-unmount) — Runs a cleanup function when the component unmounts.
- [`useIsClient`](/hooks/use-is-client) — Checks if code is running on the client (browser).
- [`useInterval`](/hooks/use-interval) — Invokes a callback repeatedly at a specified delay.
- [`useTimeout`](/hooks/use-timeout) — Handles timeouts safely in React components.
- [`useCountdown`](/hooks/use-countdown) — Manages a numeric countdown.
- [`useScript`](/hooks/use-script) — Dynamically loads external scripts.
- [`useFetch`](/hooks/use-fetch) — A wrapper around the native fetch API.

## Utilities & Events

General purpose utilities for events, refs, and themes.

- [`useEventCallback`](/hooks/use-event-callback) — Creates a memoized event callback.
- [`useEventListener`](/hooks/use-event-listener) — Adds event listeners to targets.
- [`useDebounceCallback`](/hooks/use-debounce-callback) — Creates a debounced version of a callback function.
- [`useForkRef`](/hooks/use-fork-ref) — Merges multiple refs into a single callback ref.
- [`useDarkMode`](/hooks/use-dark-mode) — Manages dark mode state.
- [`useTernaryDarkMode`](/hooks/use-ternary-dark-mode) — Manages system/light/dark modes.
