## Installation

::code-group

```bash [npm]
npx sse-hooks add use-ssr
```

```bash [yarn]
yarn dlx sse-hooks add use-ssr
```

```bash [pnpm]
pnpm dlx sse-hooks add use-ssr
```

```bash [deno]
deno run -A npm:sse-hooks add use-ssr
```

```bash [bun]
bunx sse-hooks add use-ssr
```

::

## Usage

```tsx [example.ts]
import { useSSR } from "./{hooks file}";

const { isBrowser, isServer } = useSSR();
if (isServer) {
  return <StaticLoader />;
}
return <ClientComponent />;
```

## API

### Return Value

Returns `UseSSRReturn`.

Object containing boolean flags for the current environment.
