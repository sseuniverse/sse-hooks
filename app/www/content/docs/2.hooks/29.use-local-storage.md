## Installation

::code-group{sync="pm"}

```bash [npm]
npx sse-hooks add use-local-storage
```

```bash [yarn]
yarn dlx sse-hooks add use-local-storage
```

```bash [pnpm]
pnpm dlx sse-hooks add use-local-storage
```

```bash [deno]
deno run -A npm:sse-hooks add use-local-storage
```

```bash [bun]
bunx sse-hooks add use-local-storage
```

::

## Usage

```tsx [example.ts]
import { useLocalStorage } from "./{hooks file}";

const [count, setCount, removeCount] = useLocalStorage("count", 0);
// Access the `count` value, the `setCount` function to update it and `removeCount` function to remove the key from storage.
```
