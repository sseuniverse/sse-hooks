## Installation

::code-group

```bash [npm]
npx sse-hooks add use-dark-mode
```

```bash [yarn]
yarn dlx sse-hooks add use-dark-mode
```

```bash [pnpm]
pnpm dlx sse-hooks add use-dark-mode
```

```bash [deno]
deno run -A npm:sse-hooks add use-dark-mode
```

```bash [bun]
bunx sse-hooks add use-dark-mode
```

::

## Usage

```tsx [example.ts]
import { useDarkMode } from "./{hooks file}";

const { isDarkMode, toggle, enable, disable, set } = useDarkMode({
  defaultValue: true,
});
```

## API

### Parameters

| Name    | Type              | Description                                          |
| :------ | :---------------- | :--------------------------------------------------- |
| options | `DarkModeOptions` | The initial value of the dark mode, default `false`. |

### Return Value

Returns `DarkModeReturn`.

An object containing the dark mode's state and its controllers.
