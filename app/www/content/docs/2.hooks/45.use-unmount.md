## Installation

::code-group
```bash [terminal]
npx sse-tool add use-unmount
```
```bash [npm]
npm install sse-hooks
```
::

## Usage


::code-group
```tsx [example.ts]
import { useUnmount } from './{hooks file}'

useUnmount(() => {
  // Cleanup logic here
});
```
```tsx [package.ts]
import { useUnmount } from 'sse-hooks'

useUnmount(() => {
  // Cleanup logic here
});
```
::

## API

### Parameters

| Name | Type | Description |
| :--- | :--- | :--- |
| func | `Function` | The cleanup function to be executed on unmount. |
