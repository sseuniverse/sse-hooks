## Installation

::code-group

```bash [npm]
npx sse-hooks add use-step
```

```bash [yarn]
yarn dlx sse-hooks add use-step
```

```bash [pnpm]
pnpm dlx sse-hooks add use-step
```

```bash [deno]
deno run -A npm:sse-hooks add use-step
```

```bash [bun]
bunx sse-hooks add use-step
```

::

## Usage

```tsx [example.ts]
import { useStep } from "./{hooks file}";

const [
  currentStep,
  {
    goToNextStep,
    goToPrevStep,
    reset,
    canGoToNextStep,
    canGoToPrevStep,
    setStep,
  },
] = useStep(3);
// Access and use the current step and provided helper functions.
```

## API

### Parameters

| Name    | Type     | Description                      |
| :------ | :------- | :------------------------------- |
| maxStep | `number` | The maximum step in the process. |

### Return Value

Returns `any`.

An tuple containing the current step and helper functions for navigating steps.
