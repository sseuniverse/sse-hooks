## Installation

::code-group

```bash [npm]
npx sse-hooks add use-intersection-observer
```

```bash [yarn]
yarn dlx sse-hooks add use-intersection-observer
```

```bash [pnpm]
pnpm dlx sse-hooks add use-intersection-observer
```

```bash [deno]
deno run -A npm:sse-hooks add use-intersection-observer
```

```bash [bun]
bunx sse-hooks add use-intersection-observer
```

::

## Usage

```tsx [example.ts]
import { useIntersectionObserver } from "./{hooks file}";

// Example 1
const [ref, isIntersecting, entry] = useIntersectionObserver({
  threshold: 0.5,
});

// Example 2
const { ref, isIntersecting, entry } = useIntersectionObserver({
  threshold: 0.5,
});
```

## API

### Parameters

| Name    | Type                             | Description                                |
| :------ | :------------------------------- | :----------------------------------------- |
| options | `UseIntersectionObserverOptions` | The options for the Intersection Observer. |

### Return Value

Returns `IntersectionReturn`.

The ref callback, a boolean indicating if the element is intersecting, and the intersection observer entry.
