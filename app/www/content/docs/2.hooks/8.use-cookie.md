## Installation

::code-group

```bash [npm]
npx sse-hooks add use-cookie
```

```bash [yarn]
yarn dlx sse-hooks add use-cookie
```

```bash [pnpm]
pnpm dlx sse-hooks add use-cookie
```

```bash [deno]
deno run -A npm:sse-hooks add use-cookie
```

```bash [bun]
bunx sse-hooks add use-cookie
```

::

## Usage

```tsx [example.ts]
import { useCookie } from "./{hooks file}";

// Creates a cookie named "__Secure-token"
const [token, setToken] = useCookie("token", "", {
  prefix: "__Secure-",
  secure: true,
});
```

## API

### Parameters

| Name         | Type               | Description                  |
| :----------- | :----------------- | :--------------------------- | ------------------------------- |
| key          | `string`           | The base name of the cookie. |
| initialValue | `T                 | Function`                    | The initial value of the state. |
| options      | `UseCookieOptions` | Options for customization.   |

### Return Value

Returns `any`.

A tuple containing the stored value, a setter, and a remover.
