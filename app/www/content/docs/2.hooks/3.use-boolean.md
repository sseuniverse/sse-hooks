## Installation

::code-group

```bash [npm]
npx sse-hooks add use-boolean
```

```bash [yarn]
yarn dlx sse-hooks add use-boolean
```

```bash [pnpm]
pnpm dlx sse-hooks add use-boolean
```

```bash [deno]
deno run -A npm:sse-hooks add use-boolean
```

```bash [bun]
bunx sse-hooks add use-boolean
```

::

## Usage

```tsx [example.ts]
import { useBoolean } from "./{hooks file}";

const { value, setTrue, setFalse, toggle } = useBoolean(true);
```

## API

### Parameters

| Name         | Type      | Description                                                   |
| :----------- | :-------- | :------------------------------------------------------------ |
| defaultValue | `boolean` | The initial value for the boolean state (default is `false`). |

### Return Value

Returns `UseBooleanReturn`.

An object containing the boolean state value and utility functions to manipulate the state.
