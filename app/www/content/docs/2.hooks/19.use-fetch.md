```bash [npm]
npx sse-hooks add use-fetch
```

```bash [yarn]
yarn dlx sse-hooks add use-fetch
```

```bash [pnpm]
pnpm dlx sse-hooks add use-fetch
```

```bash [deno]
deno run -A npm:sse-hooks add use-fetch
```

```bash [bun]
bunx sse-hooks add use-fetch
```

::

## Usage

```tsx [example.ts]
import { useFetch } from "./{hooks file}";

interface User {
  id: number;
  name: string;
}

const { data, loading, error, execute } = useFetch<User>(
  "https://api.example.com/user/1",
  {
    immediate: true,
    onSuccess: (data) => console.log("User loaded:", data),
  },
);
```

## API

### Parameters

| Name    | Type              | Description                                                       |
| :------ | :---------------- | :---------------------------------------------------------------- |
| url     | `string`          | The URL to fetch.                                                 |
| options | `UseFetchOptions` | Options for customizing the request and hook behavior (optional). |

### Return Value

Returns `UseFetchReturn`.

An object containing the fetched data, loading status, error, and methods to control the request.
