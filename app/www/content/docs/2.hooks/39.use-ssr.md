## Installation

::code-group
```bash [terminal]
npx sse-tool add use-ssr
```
```bash [npm]
npm install sse-hooks
```
::

## Usage


::code-group
```tsx [example.ts]
import { useSSR } from './{hooks file}'

const { isBrowser, isServer } = useSSR();
* if (isServer) {
return <StaticLoader />;
}
* return <ClientComponent />;
```
```tsx [package.ts]
import { useSSR } from 'sse-hooks'

const { isBrowser, isServer } = useSSR();
* if (isServer) {
return <StaticLoader />;
}
* return <ClientComponent />;
```
::

## API


### Return Value

Returns `UseSSRReturn`.

Object containing boolean flags for the current environment.
