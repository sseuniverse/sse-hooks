## Installation

::code-group

```bash [npm]
npx sse-hooks add use-is-mounted
```

```bash [yarn]
yarn dlx sse-hooks add use-is-mounted
```

```bash [pnpm]
pnpm dlx sse-hooks add use-is-mounted
```

```bash [deno]
deno run -A npm:sse-hooks add use-is-mounted
```

```bash [bun]
bunx sse-hooks add use-is-mounted
```

::

## Usage

```tsx [example.ts]
import { useIsMounted } from "./{hooks file}";

const isComponentMounted = useIsMounted();
// Use isComponentMounted() to check if the component is currently mounted before performing certain actions.
```

## API

### Return Value

Returns `Function`.

A function that returns a boolean value indicating whether the component is mounted.
